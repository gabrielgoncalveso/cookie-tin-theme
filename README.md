# Cookie Tin Theme

![icon](./assets/icon.png)

Celestial theme for butter cookies that love to code.

## Theme

- **Celestial Seamstress**: A fanmade theme based on the color palette of [Phase-Connect](https://phase-connect.com/)'s 2nd generation VTuber [Amanogawa Shiina](https://www.youtube.com/@AmanogawaShiina)
- **Celestial Sailor**: A purple-leaning variant of Celestial Seamstress based on Shiina's Sailor outfit
